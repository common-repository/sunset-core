This plugin was made for the <a href="https://wordpress.org/themes/my-sunset/" target="_blank">My Sunset</a> Theme and adds extra functionality to it.

== Description ==
This plugin adds custom Gutenberg blocks (such as running line, rotating text), animation options for paragraphs, headings and images, and other features.
Works best with <a href="https://wordpress.org/themes/my-sunset/" target="_blank">My Sunset</a> theme.

== Installation ==
Upload the Sunset Core plugin to your blog, activate it and start using.
Don't forget to install <a href="https://wordpress.org/themes/my-sunset/" target="_blank">My Sunset</a> theme as well.
